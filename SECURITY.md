# Безопасность

## ⚠️ ВАЖНО: Токен бота был скомпрометирован

Если вы используете токен `8259186449:AAE2p1G9_z_oD1RGN60GXYN-Z2rFRE4cBWc`, **немедленно**:

1. Перейдите в [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/revoke` для вашего бота
3. Создайте новый токен командой `/token`
4. Обновите токен в файле `.env`

## Настройка токена

Токен бота должен храниться **только** в файле `.env`, который:
- Не попадает в Git (добавлен в `.gitignore`)
- Не должен быть закоммичен
- Должен быть создан локально на каждой машине

### Создание файла .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```
TELEGRAM_BOT_TOKEN=ваш_новый_токен_здесь
```

### Запуск бота

Используйте скрипт `start_bot.ps1`, который автоматически загрузит токен из `.env`.

Или запустите напрямую:
```bash
python main.py
```
(токен будет загружен из `.env` автоматически)
