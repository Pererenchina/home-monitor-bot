# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Создание бота в Telegram

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Поиск квартир")
   - Введите username бота (должен заканчиваться на `bot`, например: `apartment_search_bot`)
4. Скопируйте полученный токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 3: Настройка конфигурации

1. Создайте файл `.env` в корне проекта
2. Добавьте в него:
```
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

Например:
```
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

## Шаг 4: Запуск бота

```bash
python bot.py
```

Вы должны увидеть сообщение:
```
INFO - Бот запущен!
```

## Шаг 5: Использование

1. Откройте Telegram и найдите вашего бота по username
2. Отправьте команду `/start`
3. Настройте фильтры командой `/filters`
4. Проверьте объявления командой `/check`

## Команды бота

- `/start` - Начать работу
- `/filters` - Настроить фильтры (комнаты, цена, арендодатель, источники)
- `/check` - Проверить новые объявления сейчас
- `/status` - Посмотреть текущие настройки

## Автоматическая проверка

Бот автоматически проверяет новые объявления каждые 5 минут и отправляет их вам, если они соответствуют вашим фильтрам.

## Решение проблем

### Бот не отвечает
- Проверьте, что токен в `.env` правильный
- Убедитесь, что бот запущен (нет ошибок в консоли)

### Не находит объявления
- Сайты могут изменить структуру - парсеры могут потребовать обновления
- Проверьте настройки фильтров - возможно, они слишком строгие

### Ошибки парсинга
- Некоторые сайты могут блокировать запросы
- Попробуйте запустить позже или обновить User-Agent в `config.py`
